# React Native Profile Card

A simple React Native application built with Expo that features a profile card, dynamic name greetings, and an interactive counter.

## 🚀 Features
- **Name Input:** Interactive `TextInput` that updates the greeting dynamically.
- **Counter:** Increase, decrease, and reset actions using `useState`.
- **Conditional Rendering:** Changes layout text based on input and count states.
